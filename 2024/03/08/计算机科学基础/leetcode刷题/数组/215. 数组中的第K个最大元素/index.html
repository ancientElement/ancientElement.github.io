Problem: 215. 数组中的第K个最大元素

解题方法

快排:

  • 判断左小于右边

    - 左右指针ij,随机数最左边的数 x = nums[i]

    - 遍历当 i < j

        - 从右往左大于x

            - j–

        - 如果i < j

            - num[i] = num[j];

            - i++;

        - 从左往右小于x

            - i++

        - 如果i < j

            - nums[j] = num[i]

            - j–;

    nums[i] = x;

    递归(nums,left,i-1);    

    递归(nums,i+1,right);
