* Properties 属性

* SubShader 子着色器

* FallBack "Diffuse" 当SubShader全部不匹配显卡时返回

  ```cs
  Shader "Hidden/NewImageEffectShader"
  {
      Properties
      {
          _MainTex ("Texture", 2D) = "white" {}
      }
      SubShader
      {
          // No culling or depth
          Cull Off ZWrite Off ZTest Always
  
          Pass
          {
              CGPROGRAM
            
              ENDCG
          }
      }
  }
  ```
