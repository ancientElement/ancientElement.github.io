## Lesson15

**InputSystem 获取任意按键**

Call会报错但是会多次监听

CallOnce只监听一次不会报错

```cs
InputSystem.onAnyButtonPress.CallOnce((ctx) =>
{
    print(ctx.path);//InputAction路径
    print(ctx.name);//InputAction名字
});
InputSystem.onAnyButtonPress.Call((ctx) =>
{
    print(ctx.path);//InputAction路径
    print(ctx.name);//InputAction名字
});
```
