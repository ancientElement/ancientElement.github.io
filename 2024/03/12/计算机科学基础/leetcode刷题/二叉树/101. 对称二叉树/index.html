Problem: 101. 对称二叉树

思路

  • 使用后序遍历,这样才能对左右节点进行一个对比

  • 先做一个总体的判断,左右是否不相等,具体为:

    - 只要有一个为空一个不为空

    - 只要两个数值不相等

  • 还有一个情况是两个子节点都为空返回true

  • 再后续遍历

  • bool a=左节点

  • bool b=右节点

  • return a && b
